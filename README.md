# Clone of firstCry

Babycare is a clone of FirstCry app is a e-commerce platform catering to all parenting needs, offering a wide range of baby products, clothing, toys, and essentials.

Tech Stacks Used:
- React Js
- Chakra UI
- Material UI
- CSS
- Cyclic
- Netlify
